# order-cancellations-dashboard

Дашборд анализа отмен заказов (Power BI)  
Мониторинг ключевых метрик отмен, выявление проблемных магазинов и анализ динамики отказов клиентов.

## Описание

Дашборд на основе данных о заказах с детализацией по магазинам, типам заказов и способам доставки.

Позволяет:
- Отслеживать общую долю отмененных заказов и динамику изменений;
- Выявлять топ-5 магазинов с наибольшим процентом отмен;
- Анализировать распределение отмен по типам заказов (Много, Самовывоз, Партнерские витрины, B2B, Быстро);
- Контролировать изменение метрик относительно предыдущего дня и прошлой недели;
- Принимать управленческие решения по снижению уровня отмен в проблемных точках.

## Ключевые метрики

| **Метрика**                                      | **Описание / Формула**                          | **Примечание**                     |
|--------------------------------------------------|--------------------------------------------------|------------------------------------|
| **Cancellation Rate (Доля отмен)**               | (Отмененные заказы / Все заказы) × 100%        | Основной показатель качества       |
| **Total Sales (Общие продажи)**                  | Суммарная выручка за период (₽)                | Включая все заказы                 |
| **Average Check (Средний чек)**                  | Выручка / Количество заказов (₽)                | Средний размер заказа              |
| **DoD Change % (Изменение к предыдущему дню)**   | ((Сегодня - Вчера) / Вчера) × 100%             | Оперативный контроль               |
| **WoW Change % (Изменение к прошлой неделе)**    | ((Эта неделя - Прошлая неделя) / Прошлая неделя) × 100% | Недельная динамика           |
| **Top 5 Stores by Cancellations**                | Магазины с наибольшим % отмен                  | Приоритетные точки для анализа     |

###### Все визуализации связаны через фильтрацию: выбор магазина/типа заказа/даты автоматически обновляет остальные элементы — анализ «от общего к частному»

## Технические детали

- **Инструмент**: Power BI Desktop (.pbix);
- **Данные**: агрегация по дате создания заказа, фильтрация по географии магазинов, типам и способам доставки;
- **Визуализации**:
  - Кольцевая диаграмма — распределение отмен по типам заказов с процентными долями;
  - Линейный график — ежедневная динамика доли отмен для выявления трендов и аномалий;
  - Горизонтальная гистограмма — топ-5 магазинов по проценту отмен;
  - Детальная таблица — полная информация по каждому заказу с возможностью drill-down;
- **KPI-карточки**: цветовая индикация изменений (красный — отрицательная динамика, зеленый — положительная);
- **Дизайн**: минималистичный интерфейс с акцентом на ключевые метрики, логотип компании.
